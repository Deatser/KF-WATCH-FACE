Инструкция по установке

  1. Скачайте файл выше на свой телефон
  2. Установите одно из приложений:
    • WearLoad (рекомендуется для Amazfit)
    • ADB App Control (универсальное)
    • Bugjaeger (альтернативный вариант)
  3. Подключите часы к телефону по Bluetooth
  4. Загрузите APK файл через приложение на ваши часы
  5. Выберите установленный циферблат в настройках часов
Важно: Циферблат работает только на умных часах с Wear OS 4.0 и выше. Для установки необходимо подключение часов к телефону.
